# AI驱动投资助手（V0.5）

基于 moomoo OpenD 的 AI 自动交易实验项目。

当前版本已经实现：

- 实时获取美股行情
- AI评分系统
- 风险控制
- 自动仓位管理
- 自动买入
- 自动卖出
- moomoo 模拟盘联动
- 审计日志记录
